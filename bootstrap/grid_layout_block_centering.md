# Grid layout使用時にdiv.row内のブロックを中央寄せにする

https://shared-blog.kddi-web.com/bootstrap4/grid.html

`div.row`に`.justify-content-center`を追加する。

```html
<div class="container">
<div class="row　justify-content-center">
  <div class="col">
    One of three columns
  </div>
  <div class="col">
    One of three columns
  </div>
  <div class="col">
    One of three columns
  </div>
</div>
</div>
```
